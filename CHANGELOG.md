# Changelog

## 1.1.1 (2026-03-09)

- Added link to [promptpocket.app](https://promptpocket.app) in the extension header
- Snippets are now automatically resolved when copying prompts
- Missing snippets prompt the user for replacement text instead of being silently skipped
- Snippets used in prompts can no longer be deleted (protection against broken references)

## 1.1.0

- Added prompt search with variable filling
- Added snippet resolution when copying prompts
- Added AES-256 encryption badge
- Added folder management (create, select)
- Added multi-language support (EN, FR, DE)
- Added dark/light/system theme support
- Added keyboard shortcut (Alt+Shift+S) for quick save
- Added context menu quick save

## 1.0.0

- Initial release
- Save prompts from any webpage
- API key authentication
- Basic prompt creation with title, content, description
- Tags and AI tools support
- Public/private visibility toggle
- Draft/published status
